## Input data

Input data has the following format:

```
RNG_seed | any
N
room_side
max_speed
min_radius
max_radius
exit_size
far_exit_distance
far_exit_size
id1 x1 y1
id2 x2 y2
id3 x3 y3
...
idN xN yN
```
